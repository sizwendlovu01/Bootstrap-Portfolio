Clone Your Future – Bootstrap Portfolio

A responsive personal portfolio website built using HTML, CSS, and Bootstrap 5. This project showcases my skills, projects, and contact information in a clean, modern layout designed to evolve as I grow in my software development journey.

The goal of this portfolio is simple:
Create a professional online presence that can be shared with recruiters, employers, and collaborators.

Think of this portfolio as a living project — every new skill, project, or improvement will be added over time, transforming it from a student showcase into a professional developer portfolio.



Project Purpose

This project was built as part of the “Clone Your Future” Bootstrap Portfolio Assignment.

The objective was to recreate a modern portfolio layout using Bootstrap 5 while ensuring:

Mobile-first responsive design

Clean UI structure

Semantic HTML

Accessibility best practices

Professional presentation suitable for job applications

Technologies Used

HTML5

CSS3

Bootstrap 5 (CDN)

Bootstrap Icons

Responsive Design Principles

No JavaScript was used, following the assignment requirement to rely on native HTML and CSS features only.
